Notification Channels ▾

Named notification channels let you send alerts to multiple destinations — multiple Discord webhooks, email addresses, or custom HTTP endpoints. Each channel has a unique name used by action rules.

{# ── Schedule ─────────────────────────────────────────────────────────────── #}
Schedule (Auto Arm/Disarm) ▾

Automatically arm at dawn and disarm at dusk. Uses 24-hour format (e.g. 06:00). Manual arm/disarm overrides the schedule until the next scheduled transition. Enable solar mode to use sunrise/sunset instead of fixed times.

{# ── Auth ─────────────────────────────────────────────────────────────── #}
Authentication ▾

Gates all web UI routes behind a login page. Pair with TLS when accessible beyond your LAN. When disabled, anyone on the network can access the dashboard and admin tools.
